Início Sobre

Esta página foi criada em uma branch separada chamada pagina-sobre.

Depois de finalizada, foi feito merge com a branch main.

Comandos utilizados

  1. git branch pagina-sobre
  2. git checkout pagina-sobre
  3. git add sobre.html
  4. git commit -m "adiciona pagina sobre"
  5. git checkout main
  6. git merge pagina-sobre
