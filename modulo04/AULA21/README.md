## Como publicar no GitHub Pages

1. Fazer push do projeto para o GitHub
2. Acessar **Settings** do repositório
3. Ir em **Pages**
4. Em **Source**, selecionar **main branch**
5. Salvar e aguardar o link gerado

## Requisitos para o GitHub Pages funcionar

- O arquivo principal deve se chamar `index.html`
- Deve estar na raiz do repositório
- Todos os caminhos de arquivos devem ser relativos
- Nomes de arquivos em minúsculas, sem caracteres especiais
